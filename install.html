AbEC - How to install



> [ Home ]





[~]$ Requirements



 
The only requirement to install and use the framework is to have the git installed in the computer. The following the command does the job.




    

    
    $ sudo apt install git

    
    








[~]$ AbEC



 
Now, to use the framework, we just need to clone the git repository on our machine. To do that, you can use the follownig command:




    

    
    $ git clone https://github.com/AbEC-EC/AbEC.git

    
    





Then, enter in the AbEC directory created and execute the install.sh script with the source command, this will install all the dependencies necessaries to run the framework. 




    

    
    $ cd AbEC

    
    





    

    
    $ source install.sh

    
    





Once installed the dependencies, we can start to use the framework going to the abec directory and executing the app.sh with the source command:




    

    
    $ cd abec

    
    





    

    
    $ source app.sh
