AbEC - How to install



> [ Home ]





[~]$ Python



 

 The requirements to install and use the framework are to have the Python 3 and git installed in the computer. The following the commands that do the job.



For the Python:





$ sudo apt install python3.10









For the Git:





$ sudo apt install git















[~]$ AbEC



 

 Now, to use the framework, we just need to clone the git repository on our machine. To do that, you can use the follownig command:






$ git clone https://github.com/AbEC-EC/AbEC.git









Once cloned, we can start to use the framework. For more information about how to use, please come here.
